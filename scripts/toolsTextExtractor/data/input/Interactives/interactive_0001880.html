BP = blood pressure; CSM = carotid sinus massage; ECG = electrocardiogram; EEG = electroencephalogram; EPS = electrophysiological study; ICD = implantable cardioverter defibrillator; ILR = implantable loop recorder; PCM = physical counter-pressure manoeuvres.
